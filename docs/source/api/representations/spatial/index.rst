#########################################
Spatial Representations API (`spatial`)
#########################################

This section of the API documentation covers featurizers that generate spatial or matrix-based representations of molecules. These representations often capture 3D structural information, atomic interactions, or graph-based connectivity in matrix form.

For a usage guide on these featurizers, please see :doc:`/usage/representations/spatial/index`.
