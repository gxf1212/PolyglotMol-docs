# Adding New Modalities

Guide for adding new data modality types to PolyglotMol.

(Content to be expanded)

## Current Modalities

- VECTOR: Traditional fingerprints/descriptors
- STRING: Raw SMILES/SELFIES
- MATRIX: 2D arrays
- IMAGE: Molecular images
- LANGUAGE_MODEL: Pre-computed embeddings

See {doc}`../architecture` for details.
